Interval Timer

ABOUT

This is a really useful interval timer
written in C++ and Allegro 5 library
by Paulina Kalicka (PAULINEK)


CREDITS

Used library: Allegro 5 (https://liballeg.org/)
Sound effects obtained from www.zapsplat.com
Icon made by Gregor Cresnar from www.flaticon.com
Font Pixel Square downloaded from kenny.com


Enjoy!
